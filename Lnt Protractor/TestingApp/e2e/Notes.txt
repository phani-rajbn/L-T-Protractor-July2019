Protractor framework is used for performing e2e testing on Angula Apps and other web centric Applications. For testing Angular Apps, U use Page object model to provide all the elements of the Page as objects and use it thro Object oriented Approach. When U provide an Testing suite, it would 2 files. one called as po.ts and other spec.ts. "po.ts" is the Page object which contains the OOP of UR Web Page. Like its fields, elements, browser navagation and many more.  The spec.ts would be a BDD File that uses Testing frameworks like Jasmine which call the required test suites.
